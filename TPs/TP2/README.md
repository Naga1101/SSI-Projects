# Correr o programa

Para realizar a criação das diretorias e executaveis
```bash
make
```
```bash
sudo make install
```

Para correr o demon basta ir a pasta bin obtida pelo comando make e correr o demon
```bash
cd bin
```
```bash
sudo ./daemon
```
Para parar o demon simplesmente matamos o processo
```bash
sudo pkill -f ./daemon
```
